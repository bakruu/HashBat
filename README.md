# HashBat
a Hash Cracker tool made with Python

```
- md4
- md5
- sha1
- sha224
- sha256 
- sha384
- sha512
```
# [ Installation ]
```
$ apt update upgrade
$ apt install python2 git
$ git clone https://github.com/ciku370/hasher
$ cd hasher
$ python2 hash.py
```
